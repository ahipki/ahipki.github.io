2015

Développeur Web

Réalisation d'un site web avec ASP.NET Web Forms à Glyndwr University (Wrexham, Pays de Galles, Royaume-Uni) pendant 16 semaines.

Learn more
Du 16 février 2015 au 5 juin 2015
L'université de Glyndwr avait besoin d'un nouveau site web gérant les écoles d'été. En effet, il existait un site web par école d'été au moment du stage. L'objectif du stage était donc de regrouper ces écoles d'été afin de pouvoir les gérer par une seule interface d'administration. Ce nouveau site web a été codé en C# en utilisant le framework ASP.NET Web Forms et Entity Framework via Visual Studio 2012.
J'ai réalisé le site web dans son intergralité en me basant sur le design du site web de l'université et en reprenant les différentes fonctionnalités existantes dans les différentes sites web de gestion d'école d'été de Glyndwr University.
  • Lieu : Département informatique de l'université de Glyndwr
  • Environnement : Windows
  • Editeur : Visual Studio 2012
  • Client : Html, Css, Javascript + jQuery
  • Serveur : Visual C#, ASP.NET Web Forms, Entity Framework, SQL Server
  • Autres outils : Google PageSpeed plugin and insights
  • Equipe : seule avec 3 intervenants (John Worden, Alison Thomas et Phil Challinor)
  • Entreprise : Glyndwr University
  • Maitre de stage : John Worden
  • Salaire : Stage non rémunéré

2014

Recherche sur la photogrammétrie

Recherche et création d'une petite application en python au LaBRI pendant 7 semaines.

Learn more
Du 7 juillet 2014 au 22 août 2014
Le premier objectif était de dresser un état de l’art sur la photogrammétrie : chercher et comprendre les méthodes pour reconstituer, à partir de plusieurs images 2D, une image en 3D et trouver des outils existants. Le second était de créer à partir d'un objet 3D des images en 2D. L'objectif final non atteint à cause d'une limite en temps et en ressources était de créer un objet 3D à partir d'images 2D. Le stage a donc aboutit à un programme sans interface graphique permettant à partir d'un objet 3D et d'un angle, de fournir l'image 2D correspondant.

J'ai aussi été invitée à faire de l'origami pour la Maison des Enfants. J'ai plié une trentaine de girafes en papier.
